# Clean-room reference policy

MoonGoons: Syndicate Rising is an original work.

The Police Chief APK may be studied to understand generic game categories, interaction patterns, pacing, and screen relationships. This repository must not contain copied APK code, textures, models, audio, dialogue, names, databases, network endpoints, encryption keys, or remotely downloaded asset bundles.

All implementation in this repository uses original MoonGoons story, terminology, characters, artwork, code, and synthesized audio.
