# MoonGoons: Syndicate Rising game design

## One-game rule

This repository contains one game and one viewpoint: the criminal side of the MoonGoons universe.

There is no campaign hub, Peacekeeper precinct, StarCraft-style battlefield, or MoonGoons MVP mode in this project.

## Structural reference

The uploaded Police Chief APK is a reference for broad structure and player flow only:

- portrait-oriented living city
- buildings that open management views
- construction and upgrade timers
- research, training, healing, and missions
- heroes and troop formations
- separate tactical battles
- a world map with marches and territory
- alliances, shops, events, rankings, and long-term progression

No APK assets, code, audio, names, dialogue, or proprietary data are included.

## MoonGoons conversion

| Reference category | Syndicate Rising system |
|---|---|
| Police precinct | Syndicate district and hideout network |
| Officers and heroes | Bosses, fixers, hackers, enforcers, runners, and sharpshooters |
| Guards, bikers, marksmen | Enforcers, runners, sharpshooters |
| Crime patrol | Heists, sabotage, extraction, smuggling, and theft |
| Arrested criminals | Captured officers, informants, rival bosses, and witnesses |
| Police technology | Black Tech and stolen Authority systems |
| Public security | Heat, secrecy, and district influence |
| World map | Lunar underworld territory map |
| Alliance | Syndicate alliance and cartel war |

## Current foundation

The current playable foundation includes the city, eight buildings, four crew members, rebuilding, upgrades, resources, Heat, Black Tech, five story scores, repeatable jobs, tactical raids, cinematics, saving, and browser/Android exports.
