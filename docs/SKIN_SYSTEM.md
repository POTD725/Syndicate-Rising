# Syndicate Rising Skin System

MoonGoons: Syndicate Rising includes five complete, selectable visual families:

1. Steel Syndicate
2. Neon Vice
3. Toxic Circuit
4. Ember Cartel
5. Cryo Authority

The selected family is applied consistently to the game rather than mixing unrelated styles. Coverage includes all hideout rooms, crew roles, resource sites, hideout defenses, Take Back threat units, mission types, communications channels, score cards, raid controls, story choices, attack cinematics, interface frames, buttons, and navigation icons.

The existing original lunar backgrounds and detailed MoonGoons artwork remain beneath the skin layer. The skin atlas supplies clear game-readable symbols and palette consistency while keeping browser and Android builds lightweight.

Use the **SKIN** control in the lunar hideout, Operations, or Communications screen to cycle through the five families. The chosen family is stored locally and restored in later sessions.
