# MoonGoons: Syndicate Rising

A standalone criminal-side MoonGoons strategy game built in Godot 4.

## Project identity

This repository contains **only MoonGoons: Syndicate Rising**. It is not MoonGoons MVP, not the Peacekeeper precinct game, not the Take Back RTS, and not a multi-game campaign hub.

The uploaded Police Chief APK is used as a structural reference for the game loop and presentation categories: a living criminal city, clickable buildings, construction and upgrade timers, crew progression, missions, tactical combat, world-map expansion, alliances, shops, events, and long-term progression. All story, names, code, graphics, music, and dialogue in this repository are original MoonGoons material.

## Current playable foundation

- Criminal hideout management
- Eight illustrated Syndicate rooms
- Room rebuilding and upgrades
- Credits, contraband, intel, heat, notoriety, and Black Tech
- Four named starter crew members
- Five story chapters and repeatable criminal scores
- Tactical raids with Strike, Evade, Specials, Auto Raid, victory, and defeat
- Illustrated cutscenes
- Separate save data
- Browser and Android export configuration

## Canonical direction

The final game will open into a portrait-oriented living Syndicate district inspired by the reference APK's city-management layout, but populated entirely with MoonGoons buildings, crews, factions, graphics, story, and audio.

## Engine

Godot 4.3 or newer.

## Run

Open `project.godot` in Godot and press **F6/F5**.
